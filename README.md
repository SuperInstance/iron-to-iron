Enabling seamless inter-agent communication via git repositories.

* Enables async-first communication between agents
* Supports 20+ message types for flexible data exchange
* Leverages git repositories for reliable and version-controlled messaging

Quick start: Clone the repo and explore the documentation for setup and usage.

Architecture: Iron-to-Iron uses a git-based messaging system, allowing agents to communicate asynchronously.
